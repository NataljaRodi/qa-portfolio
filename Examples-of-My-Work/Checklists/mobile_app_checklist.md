# Mobile Application Checklist

> Checklist demonstrating my approach to functional, usability,
performance, and reliability validation of mobile applications.

---

## Installation & Setup

**Positive**
- [ ] Application installs successfully
- [ ] Application uninstalls without errors
- [ ] Application updates correctly

**Negative**
- [ ] Installation interrupted or cancelled
- [ ] Update failure handling

---

## Authentication & User Flow

**Positive**
- [ ] Registration via email
- [ ] Social login (Google / Facebook / Apple)
- [ ] Login with existing account
- [ ] Password recovery flow

**Negative**
- [ ] Invalid login credentials show error
- [ ] Mandatory field validation enforced

---

## Navigation

**Validation**
- [ ] Menu buttons function correctly
- [ ] Back navigation works
- [ ] No dead links or inactive buttons
- [ ] Screen transitions behave correctly

---

## UI / UX

**Layout & Readability**
- [ ] UI matches expected layout
- [ ] No overlapping elements
- [ ] Text readable on small screens
- [ ] Proper alignment of components

**Orientation**
- [ ] Portrait mode display
- [ ] Landscape mode display

**Animations**
- [ ] Smooth transitions
- [ ] No visual glitches

---

## Network Behavior

**Connectivity Handling**
- [ ] Works under slow connection
- [ ] Displays error when offline
- [ ] Sync resumes after reconnection

---

## Performance

**Responsiveness**
- [ ] Acceptable startup time
- [ ] No significant lag
- [ ] Stable during regular usage

**Resources**
- [ ] Battery consumption reasonable

---

## Permissions & Security Awareness

**Permissions**
- [ ] Camera access handling
- [ ] Gallery access handling
- [ ] Microphone access handling

**Security Awareness**
- [ ] Sensitive data not exposed
- [ ] Password handling awareness checks

---

## Device Compatibility

**Coverage**
- [ ] Works across supported OS versions
- [ ] Adapts to different screen sizes

---

## Interruptions Handling

**External Events**
- [ ] Incoming call interruption
- [ ] SMS interruption
- [ ] Alarm interruption
